Instructions

1.- Execute "compile_vs2019.bat"
2.- A VS2019 solution should have been created in the "build" directory "build". This is a global solution that includes every project (in a separate project file) that we will work with through the course
3.- Double click on the solution and open the VS IDE
4.- Select the project you want to work in

Son 2 proyectos. Una librería matemática de vectores y matrices con tests respectivos y una demo de la librería.
La librería matemática contiene tests para verificar correcto funcionamiento.
La demo no es una demostración libre. El requisito es utilizar cada función de la librería de forma gráfica.
No esperar máximo sentido en el uso de cada función. Se usan contenedores de vectores propios de figuras geométricas 
para poder aplicar las transformaciones.

Incluye un visor de una base de datos SQlite programada en C++. La base de datos es modificable en tiempo de ejecución.
